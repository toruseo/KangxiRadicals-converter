# KangxiRadicals_converter

KangxiRadicals_converterは、クリップボード内の文字列に含まれる康煕部首文字を正しい日本語文字に置き換えるPythonとPowerShellのスクリプトです。このリポジトリには、Python用の`KangxiRadicals_converter.py`と、PowerShell用の`KangxiRadicals_converter.ps1`が含まれています。

## 使い方

### Python

1. Python 3がインストールされていることを確認してください。
2. pyperclipパッケージをインストールします。`pip install pyperclip`を実行してインストールしてください。
3. クリップボードに変換したいテキストをコピーしてください。
4. `KangxiRadicals_converter.py`を実行してください。
5. クリップボードに変換後のテキストがコピーされます。

### PowerShell

1. Windows PowerShellがインストールされていることを確認してください。
2. クリップボードに変換したいテキストをコピーしてください。
3. `KangxiRadicals_converter.ps1`を実行してください。
4. クリップボードに変換後のテキストがコピーされます。
